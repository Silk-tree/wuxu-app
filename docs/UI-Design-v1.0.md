# 物序 App V1.0 UI/UX 设计规范

**文档版本**: V1.0  
**创建日期**: 2026-06-25  
**产品名称**: 物序  
**设计方向**: 温暖自然 · 信息优先 · 交互愉悦 · 简洁克制

---

## 设计灵感来源

本设计从参考图「Pantry AI」的视觉语言中提取核心气质：

- **自然色调**：深绿与米白的组合，传递"新鲜"与"居家"的温暖感
- **圆润有机**：大圆角卡片与柔和阴影，消除工具的冰冷感
- **状态可视化**：参考图中冰箱内红/绿光环的过期状态表达，转化为卡片边缘色条与徽章
- **层级通透**：半透明毛玻璃效果（参考图中弹窗与底部导航），用于浮层与模态

同时严格遵循 MVP 的"简洁克制"原则，不做 3D 可视化、AI 建议等重型功能，将视觉资源集中在核心信息呈现上。

### 1.5 间距规范（8pt 网格系统）

所有间距均为 4 的倍数，基础单位为 8pt。

| Token | 值 | 用途 |
|-------|-----|------|
| `space-xs` | 4dp | 图标与文字间距、徽章内边距 |
| `space-sm` | 8dp | 卡片内部小间距、行内元素间距 |
| `space-md` | 16dp | 卡片内边距（水平）、列表项间距 |
| `space-lg` | 24dp | 卡片内边距（垂直）、区块间距 |
| `space-xl` | 32dp | 页面水平边距（左右 32dp） |
| `space-xxl` | 48dp | 大区块间距、空状态上下留白 |

**页面安全区**: 左右 32dp（`space-xl`），顶部状态栏 + 16dp，底部导航栏高度 64dp + 安全区。

### 1.6 图标风格

- **风格**: 线性图标（Outline），2px 描边
- **端点**: 圆角端点（Round Cap）

#### 2.1.3 交互状态

| 状态 | 表现 |
|------|------|
| 默认 | 卡片阴影 Level 2，背景色按状态 |
| 点击（Press） | 卡片缩放至 0.98，阴影升至 Level 3，持续 150ms |
| 释放 | 回弹至原始状态，弹性曲线 `Curves.easeOutBack`，300ms |
| 加载中 | 统计数字显示骨架屏（shimmer），卡片区域显示 3 个骨架卡片 |
| 空状态 | 隐藏统计卡片与列表，展示空状态插图和引导按钮 |
| 错误 | 页面中央展示错误图标 + "加载失败，点击重试"，点击后重新加载 |

#### 2.1.4 动画说明

| 动画 | 触发条件 | 效果 | 时长 | 曲线 |
|------|---------|------|------|------|
| 列表进入 | 页面打开 | 卡片从下向上依次滑入，间隔 50ms | 400ms | `Curves.easeOutCubic` |
| 过期呼吸 | 已过期卡片持续 | 左侧色条透明度 0.5 → 1.0 循环 | 2000ms | `Curves.easeInOutSine`，无限循环 |
| 删除滑出 | 删除确认后 | 卡片向左滑出 + 高度收缩至 0 | 300ms | `Curves.easeInCubic` |
| FAB 显隐 | 列表滚动方向变化 | 向下滚动隐藏（向下位移 80dp + 缩放 0.8），向上滚动显示 | 200ms | `Curves.easeOutCubic` |
| 统计数字 | 数据加载完成 | 数字从 0 滚动增长至目标值 | 800ms | `Curves.easeOutQuart` |

## 3. 关键交互详细说明

### 3.1 列表项左滑删除

```
滑动前:                    滑动中:                    滑动后:
+--------+                +--------+----+            +--------+----+
| 纯牛奶 |                | 纯牛奶 | 🗑️ |            | 纯牛奶 | 🗑️ |
| 06-24  |     → 左滑 →   | 06-24  |删除 |     →     | 06-24  |删除 |
+--------+                +--------+----+            +--------+----+
                         (红色背景露出)             (保持展开，等待点击)
```

**交互规则**

- 触发阈值：水平位移超过卡片宽度的 30%（约 100dp）
- 滑动阻力：超过阈值前阻尼系数 0.6，超过后弹性回拉
- 背景层：红色 `#E05252`，居中显示白色垃圾桶图标 24dp + "删除" C1 白色
- 点击删除：弹出确认框（底部弹窗），文案"确定删除「{名称}」吗？删除后不可恢复"
- 确认后：卡片向左滑出屏幕，高度收缩至 0，下方卡片自动补位
- 取消：松手后若未达阈值，卡片弹性回弹至原位（弹簧动画）

**动画参数**

| 阶段 | 效果 | 时长 | 曲线 |
|------|------|------|------|
| 滑动 | 跟随手指，实时响应 | — | — |
| 回弹 | 卡片回到原位 | 300ms | `Curves.easeOutBack` |
| 展开保持 | 背景层随卡片同步移动 | — | — |
| 删除滑出 | 卡片向左滑出 + 高度收缩 | 300ms | `Curves.easeInCubic` |
| 下方补位 | 下方卡片向上滑动补位 | 300ms | `Curves.easeOutCubic` |

**触发条件**: 表单校验通过，数据成功写入数据库

| 首页 | `Scaffold` + `CustomScrollView` + `SliverAppBar` + `SliverList` | 统计卡片用 `SliverToBoxAdapter`，列表用 `SliverList` 实现瀑布流效果 |
| 首页卡片 | `Dismissible` (左右滑) 或 `flutter_slidable` | 左滑删除、右滑编辑，推荐 `flutter_slidable` 更灵活 |
| 空状态 | `AnimatedSwitcher` + `Center` | 列表与空状态切换时带动画过渡 |
| 添加页 | `Scaffold` + `SingleChildScrollView` + `Padding` | 表单内容较少，无需复杂滚动 |
| 日期选择 | `showModalBottomSheet` + `CupertinoDatePicker` / `showDatePicker` | iOS 用 Cupertino 风格，Android 用 Material |
| 详情页 | `Scaffold` + `Hero` + `ListView` | 大卡片使用 Hero 动画与首页卡片衔接 |
| 设置页 | `Scaffold` + `ListView` + `SwitchListTile` | 标准列表页布局 |
| 底部弹窗 | `showModalBottomSheet` + `ClipRRect` (圆角) | 统一圆角 24dp 弹窗风格 |

### 4.2 动画方案推荐

| 动画场景 | 推荐方案 | 代码提示 |
|---------|---------|---------|
| 页面转场（首页 → 详情） | `Hero` + `PageRouteBuilder` | `Hero(tag: 'item_${item.id}', child: Card(...))` |
| 列表项入场 | `AnimatedList` 或 `SliverAnimatedList` | `AnimatedList.insertItem(index, duration)` |
| 卡片呼吸效果 | `AnimationController.repeat(reverse: true)` | `Tween<double>(begin: 0.5, end: 1.0).animate(controller)` |
| FAB 显隐 | `AnimatedOpacity` + `AnimatedSlide` | 监听 `ScrollController` 方向变化 |
| 统计数字滚动 | `TweenAnimationBuilder<int>` | `Tween(begin: 0, end: target).animate` |
| 开关切换 | `AnimatedContainer` | 背景色 + 滑块位置同时过渡 |
| 空状态插图浮动 | `TweenAnimationBuilder<double>` + `Transform.translate` | 正弦波计算 Y 轴偏移 |
| 保存按钮状态 | `AnimatedContainer` | 宽度/颜色/文字的平滑过渡 |
| 页面底部滑入 | `showModalBottomSheet` + `DraggableScrollableSheet` | 添加页作为 BottomSheet 实现 |
| 删除项收缩 | `AnimatedContainer` | 高度从实际高度动画到 0 |

### 4.3 状态管理建议

| 场景 | 推荐方案 |
|------|---------|
| 全局物品列表 | `ChangeNotifier` + `Provider` 或 `Riverpod` |
| 付费状态 | `SharedPreferences` + `ChangeNotifier` |
| 通知开关 | `SharedPreferences` + `ChangeNotifier` |
| 表单状态 | `StatefulWidget` + 局部 `TextEditingController` |
| 列表操作（增删改） | 通过 `ItemProvider` 统一暴露 `addItem` / `updateItem` / `deleteItem` 方法 |
